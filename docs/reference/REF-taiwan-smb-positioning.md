## Part 7.5 — 台灣 SMB 市場定位（2026-03-23 洞察）

### 痛點不是「找不到文件」，是「流程跑不動」

台灣中小企業需要 ERP 等級的資訊流通，但買不起也不想導入傳統 ERP。

### 敘事

- **不說**：「我們做知識圖譜」「我們做 AI context 層」
- **說**：「你不用買 ERP，AI 就能幫你跑流程」「每個員工和 AI 都知道公司在幹嘛」

底層還是 ontology + MCP + AI agents，但老闆聽到的是流程變快了。

### ZenOS 的賭注

如果未來是 Copilot 模式（人問 AI 答）→ Glean 贏
如果未來是 Agent 模式（AI 自己做事）→ Glean 不夠，需要 ZenOS

ZenOS 賭 agent 時代。
