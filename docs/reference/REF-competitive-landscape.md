## Part 7.4 — 競品定位（2026-03-23 研究）

### ZenOS 的位置是空的

沒有產品同時滿足：AI 自動建構 ontology + SMB 導向 + 透過 MCP 給 AI agents 用。

### 最接近的競品

| 競品 | 差距 |
|------|------|
| **Graphlit** | 技術最近（Context Layer for AI Agents, 有 MCP）。但是開發者基礎設施，不是產品。沒有治理方法論 |
| **Tana** | 精神最近（知識圖譜，$25M 融資）。但是個人工具，要求搬進 Tana，沒有 MCP |
| **Glean** | 企業搜尋（$7.2B），不是知識圖譜。$50K 起跳，不會下沉 SMB |
| **Palantir** | Ontology 先驅，但只做結構化資料，$5M+/yr |
| **Notion/Guru** | 文件倉庫 + 搜尋，不是 ontology。反而是 ZenOS 的資料來源 |

### 護城河

技術不是護城河。**方法論才是**：四維標籤、漸進式信任、雙層治理、對話式建構。

Foundation Capital 2025 年底發表「Context Graphs 是兆元機會」論文，驗證賽道方向。窗口 12-18 個月。
