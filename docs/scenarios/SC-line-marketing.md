# ZenOS LINE Agent — 行銷場景

> 主體：一間台灣中小企業（以「豐禾數位」為例，10-30 人規模的 B2B 軟體服務公司）
> 情境：他們有 LINE 群組，有行銷需求，但行銷知識散落各處

---

## 背景

豐禾數位有三條產品線，行銷由老闆直接主導，有一個兼職行銷夥伴。
公司 LINE 群組是日常溝通的主要場所。

ZenOS LINE agent（以下簡稱「小禾」）被加入公司的行銷群組。

---

## 場景一：行銷方向決策捕獲

**群組：** 行銷討論群

```
老闆：這季我們主打製造業客群，
     訴求要放在「縮短交期」不要再打功能了，
     功能大家都差不多

小禾：老闆，我把這個記下來：
     「Q2 行銷方向：目標客群 製造業 / 核心訴求 縮短交期 / 停用訴求 功能比較」
     這樣對嗎？

老闆：對，加一條：不要提競業名字

小禾：好，補上了。
```

**之後行銷夥伴開始做素材，直接問小禾：**

```
行銷夥伴：這次 EDM 的核心訴求是什麼？

小禾：根據老闆上週定的方向：
     核心訴求「縮短交期」，目標製造業，不提功能比較，不提競業。
     要我把完整 brief 整理給你嗎？
```

---

## 場景二：文案審核 → 品牌規則沉澱

**群組：** 行銷討論群

```
行銷夥伴：[傳一段廣告文案]
         「導入豐禾系統，讓你的訂單從接單到出貨快 40%」

老闆：數字不要寫這麼死，改成「大幅縮短」
     還有「系統」換成「平台」，系統聽起來很舊

行銷夥伴：好

[小禾在背景捕獲]
→ 數字規則：不用具體百分比，用「大幅」「顯著」
→ 用語規則：「系統」→「平台」
→ 核准文案方向：強調交期縮短
```

**三個月後，換了新的文案寫手：**

```
新文案：[傳草稿，裡面寫「效率提升 50%」]

小禾：提醒一下，根據之前的規則，
     豐禾的文案不使用具體數字，
     建議改成「大幅提升效率」。
     另外「系統」這個詞老闆偏好用「平台」。
```

---

## 場景三：行銷夥伴即時 onboarding

**群組：** 行銷討論群，新夥伴加入第一天

```
新行銷夥伴：你好，我先了解一下公司的產品和目標客群

@小禾 幫我整理一份行銷背景給新夥伴

小禾：好的，豐禾數位行銷背景如下：

【產品線】
- 供應鏈管理平台（主力產品）
- 報表自動化模組（附加銷售）
- 導入顧問服務（高毛利）

【目標客群】製造業、50-500 人規模、有在用 ERP 但報表還是靠 Excel 的公司

【當前行銷方向】Q2 主打縮短交期訴求，針對製造業採購/IT 主管

【品牌語氣】專業但不艱澀，不用技術名詞
【用語規則】「平台」不是「系統」／不寫具體數字／不提競業

需要補充什麼嗎？
```

## 場景四：客戶反饋累積

**群組：** 業務 + 行銷群

```
業務 A：今天去拜訪客戶，他說「你們這個跟 SAP 有什麼不一樣？」
        我說我們比較便宜輕量，他說「那功能是不是比較少？」
        感覺說不過他

業務 B：我上週也遇到一樣的問題

@小禾 記一下，客戶一直問跟 SAP 的差異

小禾：記下來了。目前累積的客戶常見問題：
     1. 「跟 SAP 有什麼不同？」（第 3 次）
     2. 「導入要多久？」（第 2 次）
     3. 「可以跟我們現有系統整合嗎？」（第 2 次）

     「跟 SAP 的差異」已經被問 3 次了，
     要不要請行銷準備一個標準說法？

老闆：對，這個要處理，行銷你來準備一個一頁說明
```

## 場景五：行銷跟產品不同步（最常出事的地方）

**群組：** 行銷討論群

```
行銷夥伴：下週要發一篇文章，主題是我們的「AI 自動報表」功能上線了

小禾：等一下，根據目前的產品資料，
     「AI 自動報表」的狀態是「測試中，預計 Q3 正式上線」。

     如果現在就對外宣傳，可能跟實際時間有落差。
     要繼續還是等功能確定後再發？

行銷夥伴：啊，我以為已經上了，謝謝
         我去確認一下再說
```

---

## 這五個場景解決的共同問題

| 問題 | 傳統做法 | ZenOS LINE agent |
|------|---------|-----------------|
| 行銷方向沒有紀錄 | 靠人記憶、翻 LINE 歷史 | 決策當下捕獲，永久可查 |
| 每次 brief 都要重來 | 老闆花時間解釋 | @小禾 即時輸出完整背景 |
| 用語規則靠人記 | 改稿時才發現 | 自動提醒，新人也適用 |
| 客戶反饋沒有累積 | 各自散落在業務腦袋裡 | 自動彙整，自動提醒行動 |
| 行銷跟產品說法不一 | 出事才發現 | LINE 裡即時攔截 |

---

## 對老闆的核心承諾

> 你在 LINE 裡說的每一個行銷決策，都會自動成為下一次的起點。
> 不用再跟新夥伴解釋品牌語氣是什麼、這季主打什麼客群、什麼說法不能用。
> 說過一次，豐禾的每一個人和每一個 AI 工具，都會知道。
